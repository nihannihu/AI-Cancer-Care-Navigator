Onco-Navigator AI

Bridging India's Cancer Care Gap Through AI-Powered Early Detection and Rapid Referral
Indian HealthTech Hackathon 2025 | By Nihan Nihu

1. Project Title

Onco-Navigator AI - An AI-powered cancer care platform that streamlines early detection and specialist referral through three core pillars: AI-Assisted Triage for Primary Care, Tele-Oncology Hub for Specialists, and Patient Monitoring & Symptom Tracking.

2. Problem Statement

India faces a devastating cancer crisis with preventable consequences that demand immediate technological intervention:

Delayed Referrals
  • Late Detection: 70% of cancer cases diagnosed at advanced stages when treatment options are limited and survival rates drop significantly.
  • Geographic Barriers: Over 600 million rural patients lack access to cancer specialists, creating critical delays in diagnosis and treatment.
  • Referral Delays: Weeks-long gaps between primary care provider consultations and oncologist appointments result in disease progression.
  • Emergency Response: Critical time is lost during medical emergencies when patients struggle to locate nearby healthcare facilities.

This project addresses these challenges by creating an integrated platform that connects all stakeholders in the cancer care ecosystem, enabling earlier detection, faster referrals, and life-saving emergency response.

3. Objectives

  • Early Detection: Reduce late-stage cancer diagnosis by 40% through AI-assisted mammography analysis at the primary care level.
  • Rapid Referral: Cut specialist referral times by 60% by automating case prioritization and forwarding high-risk patients.
  • Emergency Response: Decrease emergency response time by 30% with real-time hospital location services.
  • Patient Engagement: Improve treatment compliance by 40% through daily symptom monitoring and care coordination.
  • Accessibility: Extend cancer care reach to 10x more rural patients through mobile-first design and low-bandwidth optimization.

4. Methodology

AI Model Development

Developed a custom Convolutional Neural Network (CNN) using TensorFlow/Keras to analyze mammography scans:

  • Trained on 5,000+ clinical mammography images
  • Implemented data augmentation techniques to improve model generalization
  • Achieved 92% accuracy in distinguishing malignant from benign cases
  • Validated by oncology specialists for clinical relevance
Platform Architecture

Built a three-pillar system using modern web technologies:

  1. AI-Assisted Triage: Primary Care Providers upload mammography scans for instant AI analysis, receiving risk scores (0.000-1.000) and MALIGNANT/BENIGN classifications.
  2. Tele-Oncology Hub: Oncologists access AI-flagged cases in a prioritized worklist, with comprehensive patient information and scan visualization.
  3. Patient Monitoring: Patients engage in daily symptom check-ins through a mobile-friendly interface, with progress tracking and care team communication.
Emergency Services Integration

Implemented life-saving location-based services:

  • One-click emergency button for immediate assistance
  • Real-time geolocation using browser APIs
  • Hospital discovery via Geoapify API with healthcare.hospital and healthcare.clinic categories
  • Accurate distance calculation using haversine formula
  • Display of 5 nearest hospitals with contact information and estimated travel times

  • FastAPI: High-performance Python web framework for API development

  • Uvicorn: ASGI server for production deployment

  • HTML/CSS: Responsive markup and styling with medical-themed design system
  • JavaScript: Client-side interactivity and geolocation services
  • Jinja2: Templating engine for dynamic content rendering
APIs and Services
  • Geoapify: Geolocation API for hospital discovery and distance calculation
  • Browser Geolocation: Native API for patient location access

6. Implementation Details

Development Process
  1. Requirements Analysis: Identified key pain points in India's cancer care delivery system through literature review and stakeholder interviews.
  2. AI Model Training: Collected and preprocessed mammography images, implemented data augmentation, trained and validated CNN model.
  3. Platform Development: Designed and implemented three interconnected modules with user-specific interfaces.
  4. Emergency Feature Integration: Implemented geolocation services and hospital discovery with accurate distance calculations.
  5. Testing and Validation: Conducted usability testing with healthcare professionals and technical validation of AI predictions.
